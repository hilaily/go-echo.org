CRUD 示例

服务端

server.go





客户端



curl



创建 User



curl -X POST \
  -H 'Content-Type: application/json' \
  -d '{"name":"Joe Smith"}' \
  localhost:1323/users




Response



{
  "id": 1,
  "name": "Joe Smith"
}




获取 User



curl localhost:1323/users/1




Response



{
  "id": 1,
  "name": "Joe Smith"
}




更新 User



curl -X PUT \
  -H 'Content-Type: application/json' \
  -d '{"name":"Joe"}' \
  localhost:1323/users/1




Response



{
  "id": 1,
  "name": "Joe"
}




删除 User



curl -X DELETE localhost:1323/users/1




Response



NoContent - 204
